## 用法

```
Usage: cc0 [options] input

Positional arguments:
input           Source code file[Required]

Optional arguments:
-h --help       show this help message and exit
-c              Binary
-s              Assembly
-o --output     Output file
```

未指定输出文件名时，默认为`out`
